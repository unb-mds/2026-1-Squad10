    Fast API

    1. O que é Fast API?

    • Um framework moderno e de alta performance em python focado no desenvolvimento de API’s
    • Moderno, rápido e simples
    • Sua primeira versão foi lançada no dia 15 de Novembro de 2018

    2. Principais Características

    Rápido:

    • API’s desenvolvidas com o FastAPI possuem uma alta performance
    • É considerado um dos frameworks Python mais rápidos, perdendo apenas para o Starlette e o Uvicorn, que são frameworks utilizados internamente pelo FastAPI
    • Utiliza também o Pydantic (para a parte de dados), garantindo rapidez excepcional

    Intuitivo:

    • Seu código fonte foi inteiramente desenvolvido utilizando o recurso de type hints do Python, possibilitando que as IDE’s e Editores de Código Fonte possuam um melhor intellisense
    • Menos gasto de tempo debugando o código

    Fácil:

    • Foi inteiramente pensado para ser fácil de usar e aprender
    • Menos gasto de tempo lendo a documentação
    • Baseia-se nas dicas de tipo padrão do Python, o que permite validação automática de dados e melhor suporte dos editores de código (como o autocomplete).

    Robusto:

    • O código desenvolvido já está pronto para produção, assim não precisamos fazer nenhuma alteração para então colocar as aplicações desenvolvidas no ar
    • O FastAPI gera a documentação utilizando o Swagger e o OpenAPI de forma automática
